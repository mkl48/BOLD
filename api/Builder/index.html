Builder

A fluent message builder, returned by Bold.Compose. It accumulates a text string in Bold's markdown plus structured embeds, then :Send()s it.

Inline text verbs (each appends and returns self for chaining): :Text, :Bold, :Italic, :Underline, :Strike, :Code, :Spoiler, :Emoji, :Quote, :Space, :Line. Colour, links, mentions, panels and embeds have their own methods below. 

Bold.Compose()
    :Bold("Round over!"):Line()
    :Text("winner "):Mention(winner)
    :Send()

Link​

</>

Mention​

</>
Builder:Mention(playerOrName: Player | string) → ()

Appends an @mention of a player (or a raw name).

Panel​

</>
Builder:Panel(
name: string,
args: {[string]: any}?,
body: string?
) → ()

Appends a :::name … ::: panel fence for a renderer registered with Bold.RegisterPanel. args become the panel's key/value arguments.

Embed​

</>
Builder:Embed(embed: Types.Embed) → ()

Attaches an embed (from Bold.Image, Bold.Action, etc.).

Kind​

</>
Builder:Kind(kind: string) → ()

Sets the message kind (chat, system, announcement, whisper, team, or your own).

Reply​

</>
Builder:Reply(message: any) → ()

Marks this message as a reply to another, showing a quoted preview.

To​

</>
Builder:To(player: Player) → ()

Whispers the message to a single player (delivered to them and echoed to the sender).

Payload​

</>
Builder:Payload() → any

Returns the assembled { text, embeds, kind, tags, reply_to, recipient_id, pinned } payload without sending it.

Send​

</>
Builder:Send() → ()

Sends the assembled message through Bold's pipeline.
