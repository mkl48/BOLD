# Using the chat (players)

This page is written for **players** — everything you can type into a Bold chat. Devs can link players straight here.

## Formatting your messages

Wrap text in these to style it:

| Type this | You get |
| --- | --- |
| `**bold**` | **bold** |
| `*italic*` | *italic* |
| `__underline__` | underline |
| `~~strike~~` | ~~strikethrough~~ |
| `` `code` `` | `inline code` |
| `\|\|secret\|\|` | a spoiler — click to reveal |
| `{red\|hi}` or `{#ff8800\|hi}` | coloured text |
| `[click](https://…)` | a link |
| `> quoted` | a quote block |
| `# Title` | a big heading (`#`, `##`, `###`) |

## Mentioning people

Type `@` and start typing a name — a list pops up, pick with the arrow keys and **Tab/Enter**. Mentioning someone highlights the message for them and pings them.

```
nice shot @Builderman!
```

## Emoji & stickers

Type `:` then a name (`:fire:`, `:tada:`) and pick from the list. Stickers show up in the same menu.

## Spoilers

Hide something with `||…||`. It shows as a covered block until someone clicks the message to reveal it.

```
the boss is ||actually a robot||
```

## Reactions & replies

Hover a message to get its toolbar:

- **☺** — react with an emoji.
- **↩** — reply. Your message shows a little quote of theirs.

## Commands

Start a message with `/`:

| Command | What it does |
| --- | --- |
| `/me <action>` | `* You wave` |
| `/shrug` | adds ¯\\\_(ツ)\_/¯ |
| `/poll Question \| A \| B` | starts a poll with clickable options |
| `/incog <player> <message>` | private whisper to one player |
| `/ignore <player>` | hide someone's messages for you |
| `/unignore <player>` | stop ignoring them |
| `/clear` | clear your own chat |
| `/help` | list every command |

For `/incog` and `/ignore`, start typing a name after the command and pick from the popup.

## Polls

`/poll` builds a little poll card with clickable options:

```
/poll Best map? | Ruins | Bunker | Rooftop
```

Everyone sees the poll and can tap an option.

---

Long messages collapse with a **Show more** link, and older messages fade — hover them to bring them back.
